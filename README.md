# Java   

## 컬렉션 프레임워크   
### List   
  * ArrayList   
    * 장점 : 저장된 인스턴스의 참조가 빠르다      
    * 단점   
        * 저장 공간을 늘리는 과정에서 시간이 비교적 많이 소요된다   
        * 인스턴스의 삭제 과정에서 많은 연산이 필요하여 느릴 수 있다   
  * LinkedList   
    * 장점   
        * 저장 공간을 늘리는 과정이 간단하다   
        * 저장된 인스턴스의 삭제 과정이 단순하다   
    * 단점 : 저장된 인스턴스의 참조 과정이 배열에 비해 복잡하여 느릴 수 있다
